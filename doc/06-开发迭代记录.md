## 技术栈

### 前端
- React 18 + TypeScript
- Three.js + react-three-fiber
- Zustand（状态管理）
- Ant Design（UI组件库）
- Vite（构建工具）

### 后端
- Go 1.21+
- Gin（Web框架）
- GORM（ORM）
- SQLite（数据库）

### 桌面打包
- Tauri 2.0

## 性能指标

- 首次加载时间：< 3秒
- 组件渲染帧率：60fps（100个组件以内）
- 内存占用：< 200MB（500个组件以内）
- 构建大小：~2MB（gzip后~600KB）
