### 使用指南

#### 启动应用

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

#### 基本操作

1. **添加组件**
   - 从左侧组件库点击组件
   - 或拖拽组件到3D视图

2. **选择组件**
   - 单击选择单个组件
   - Shift+单击多选组件
   - 框选多个组件

3. **移动组件**
   - 选中组件后拖拽
   - 使用属性面板精确调整位置

4. **旋转组件**
   - 使用属性面板旋转
   - 或使用快捷键旋转

5. **删除组件**
   - 选中组件后按Delete键
   - 或使用工具栏删除按钮

#### 快捷键

| 快捷键 | 功能 |
|--------|------|
| V | 选择工具 |
| M | 移动工具 |
| R | 旋转工具 |
| G | 切换网格显示 |
| L | 切换连接点显示 |
| 1 | 真实感模式 |
| 2 | 线框模式 |
| 3 | X光模式 |
| 4 | 黑白模式 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+D | 复制选中组件 |
| Ctrl+A | 全选 |
| Delete | 删除选中组件 |
| Escape | 取消选择 |

#### 文件操作

1. **保存设计**
   - 点击"保存"按钮
   - 输入设计名称
   - 设计保存到本地存储

2. **加载设计**
   - 点击"打开"按钮
   - 从列表中选择设计
   - 设计加载到编辑器

3. **导出设计**
   - 点击"导出"按钮
   - 选择导出格式（KCD、PNG、OBJ）
   - 下载导出文件

4. **材料清单**
   - 切换到"成本"标签页
   - 点击"计算"按钮
   - 查看材料清单和成本
   - 导出或打印清单

#### 结构分析

1. **执行分析**
   - 切换到"分析"标签页
   - 点击"分析"按钮
   - 查看分析结果

2. **查看结果**
   - 稳定性分数
   - 问题列表
   - 改进建议
   - 结构统计

3. **优化设计**
   - 根据建议调整设计
   - 重新分析验证

### 技术规格

#### 性能指标

- **首次加载时间**：< 3秒
- **组件渲染帧率**：60fps（100个组件以内）
- **内存占用**：< 200MB（500个组件以内）
- **构建大小**：~2MB（gzip后~600KB）

#### 支持平台

- **操作系统**：Windows、macOS、Linux
- **浏览器**：Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **屏幕分辨率**：1024x768 及以上

#### 组件规格

- **管子**：直径2.5cm，长度10/15/30/60cm
- **弯头**：角度45/90度，直径2.5cm
- **三通/四通**：直径2.5cm
- **平台**：尺寸30x30/60x60/90x90/120x60cm
- **附件**：秋千、滑梯、绳梯、攀爬墙、猴杆

---

### 已知问题

1. **性能问题**
   - 大量组件（>500）时可能卡顿
   - 复杂场景渲染较慢

2. **功能限制**
   - 组件自动连接不够智能
   - 部分组件类型缺失

3. **兼容性问题**
   - 部分旧版浏览器不支持
   - 移动端体验较差
