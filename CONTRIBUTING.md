# 贡献指南

感谢你对 Kid Climber 项目的关注！

## 如何贡献

### 报告问题

1. 使用 [Issue Tracker](https://github.com/YOUR_USERNAME/kid-climber/issues)
2. 描述问题的详细步骤
3. 提供错误截图（如有）
4. 说明你的环境信息

### 提交代码

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

### 开发规范

- 使用 TypeScript 进行前端开发
- 遵循 ESLint 代码规范
- 提交前运行测试
- 保持提交信息清晰

### 代码风格

- 使用 2 空格缩进
- 使用单引号
- 使用分号
- 遵循项目现有的代码风格

## 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 运行测试
npm test
```
